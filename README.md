## Overview

MeetingAI is an application designed to enhance team collaboration by efficiently managing meeting summaries, tasks, and reports. This app helps streamline communication and keep all team members on the same page.

## Features

- **Meeting Summaries**: Automatically generate summaries for meetings, capturing essential points and discussions.
- **Task Management**: Assign and manage tasks related to discussions and decisions made in meetings.
- **Report Generation**: Generate detailed reports to track meeting outcomes and team performance over time.
- **Team Collaboration**: Facilitate ongoing discussions and collaborations through integrated tools.

## Installation

To install MeetingAI, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/dimitrisabra/meetingai-assistant.git
   ```
2. Navigate to the directory:
   ```bash
   cd meetingai-assistant
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```

## Usage

After installation, you can start the application by running:
```bash
npm start
```
